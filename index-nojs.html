Simple Web Music Player — No JavaScript

This version does not use JavaScript. Edit the src attribute on each audio element below to point to your local audio file (relative path like assets/song1.mp3 or an absolute file URL like file:///C:/path/to/song.mp3), then click Play on the control to play that song.
